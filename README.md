# Sales Intelligence Dashboard

An AI-powered business analytics dashboard built using Streamlit, Pandas, Plotly, and Google Gemini.

## Features

- Interactive sales analytics dashboard
- City and month filtering
- Revenue and product insights
- AI Sales Assistant powered by Google Gemini
- Executive Report Generator
- Investor Summary Generator
- Downloadable business reports

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Google Gemini API

## Dataset

Sales transaction dataset containing:
- Revenue
- Orders
- Products
- Cities
- Monthly trends

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Highlights

- AI-powered business question answering
- Automated executive reporting
- Investor-focused business summaries
- Interactive visual analytics
